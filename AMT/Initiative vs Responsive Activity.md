We can determine if activity is **initiative or responsive** by comparing the relationship of the day's structure to the **previous day's value area.**

**Initiative buying** 
- Buying activity occurring **within or above** the previous day's value area.
**Initiative selling**
- Selling activity occurring **within or below** the previous day's value area.
**Responsive buying**
- Buying activity when price is **below** previous day's value area.
**Responsive selling**
- Selling activity when price is **above** the previous day's value area.

**Initiative buying and responsive selling** TPOs occur **above** the previous day's **value area**. **Responsive buying and initiative selling** TPOs occur **below** the previous days value are.

Any TPOs activity occurring within the **previous days** value area is considered **initiative**.
